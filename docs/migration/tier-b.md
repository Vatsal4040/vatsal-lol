# Tier B Games Migration & Polish Workflow

## Scope
Tier B games require minor polish (touch control repairs, spacing, alignment, mobile fixes, performance pacing) in addition to V4 integration. Gameplay mechanics remain unchanged.

## Standard Tier B Iterative Workflow
```
Audit
  ↓
Issue Report
  ↓
Proposed Fix
  ↓
User Approval
  ↓
Implementation
  ↓
Regression Testing
  ↓
Verification
  ↓
Done
```

## Game List & Primary Focus
- `2048`: Desktop score layout; mobile swipe gesture lock fix.
- `adjustme`: Repair controls (enable drag & tap, erase mode).
- `bubbles`: Dual-font refresh flash fix.
- `bugsmash`: High-Hz refresh rate pacing fix.
- `draw-a-circle`: Chalk-like slate visual touch-up.
- `future-timeline`: SVG positioning & mobile animation fix.
- `not_scary`: Fix desktop & mobile alignment bugs.
- `onelightday`: Scroll & ending transition polish.
- Additional Tier B games (`emojis-2-movies`, `everything-is-progressing`, `flash-memory`, `focus`, `guess-the-lie`, `hardword`, `jokes-if-you-handle`, `lets-settle`, `luckorpredict`, `mastermind`, `paddleclub`, `snake`, `soundbar`, `standing`, `tower-of-hanoi`, `which-number`, `wordle`, `would-you-press-the-button`, `xo`, `your-life-in-numbers`).
